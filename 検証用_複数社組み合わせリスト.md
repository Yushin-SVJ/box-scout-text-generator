## 「複数社検討シート」へのコピペ用データ

以下を「複数社検討シート」のA列・B列に貼り付けてください（ヘッダー行の下から）：

```
SalesMarker	SmartHR
アンドパッド	Sansan
SmartHR	Sansan
Helpfeel	匠技研
ユーザベース	SmartHR
kubell	アンドパッド
SalesMarker	ラクス
kubell	ユーザベース
```

## 検証チェックリスト

生成された複合スカウトについて以下を確認：

- [ ] **構造の選択理由が明確か**（pattern_reason に記載）
- [ ] **2社の共通点・違いが整理されているか**
- [ ] **件名が複数社スカウトとして適切か**（企業名の羅列ではなく、共通軸を示しているか）
- [ ] **本文が冗長になっていないか**（単社×2ではなく、統合された1通になっているか）
- [ ] **固定フッターが正しく挿入されているか**
- [ ] **A/B/C 各構造がバランスよく生成されるか**（偏りがないか）
- [ ] **モード1（情熱）とモード2（市場分析）の表現が適切に使い分けられているか**

| 複合パターン | 出現が期待される組み合わせ |
